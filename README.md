# 🤖 Trilha de Desenvolvimento de IA

Este repositório foi criado com o objetivo de reunir **anotações, aprendizados e práticas** ao longo da Trilha de Desenvolvimento de Inteligência Artificial da [Rocketseat](https://www.rocketseat.com.br/). Aqui estão registrados os principais conceitos explorados durante a formação, bem como códigos e insights que ajudam a consolidar o conhecimento.

## 📚 Conteúdo Estudado

- **Introdução**
  - Visão geral da trilha e ferramentas utilizadas

- **Estatística para Devs**
  - Fundamentos estatísticos aplicados à IA: média, mediana, variância, distribuição normal, entre outros

- **Análise Exploratória de Dados**
  - Manipulação de dados com Pandas
  - Visualizações com Matplotlib e Seaborn

- **Aprendizado de Máquina**
  - Conceitos de treino e teste, overfitting, underfitting, validação cruzada e métricas de avaliação

- **Algoritmos Supervisionados**
  - Modelos como regressão linear, regressão logística, árvores de decisão e k-NN

- **Algoritmos Não Supervisionados**
  - Técnicas de agrupamento como K-Means, DBSCAN e PCA

- **Ensemble de Modelos**
  - Métodos como Random Forest, Gradient Boosting e Stacking

- **Desafio Final**
  - Projeto prático integrando todos os aprendizados da trilha
